# JavaScript Learning Hub – Part 2
JavaScript Learning Hub – Part 2 is an interactive learning website designed to help beginners understand important JavaScript concepts through simple explanations and live examples.
## 🚀 Features
- DOM Manipulation Examples
- JavaScript Objects
- JavaScript Classes
- Event Handling
- Student Grade Calculator
- Interactive "Run Example" Buttons
- Beginner-friendly UI
## 📚 Concepts Covered
- Document Object Model (DOM)
- Object Creation & Methods
- ES6 Classes
- JavaScript Events (Click, Hover, Input)
- Conditional Statements
- Dynamic Web Interaction
## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
## 🎯 Purpose
This project is created as a mini learning platform for students to understand JavaScript concepts with real working examples.
